# ⚽ World Cup Prediction

A football prediction web application built using **HTML, CSS, and Vanilla JavaScript**.

This project allows users to predict match scores for the FIFA World Cup while also comparing their predictions with a simple AI-powered prediction bot based on historical team strengths.

## 🚀 Features

- View World Cup fixtures
- Predict match scores
- AI Bot match predictions
- Score calculation system
- Dynamic fixture rendering using JavaScript
- Responsive and clean UI
- Built without any external libraries

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
